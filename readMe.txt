Optimize your WooCommerce store for maximum performance and speed with Woo Performance Optimizer Lite.

== Description ==

Woo Performance Optimizer Lite is a lightweight plugin designed to improve your WooCommerce store's performance through various optimization techniques.

**Features:**
* Database query optimization
* Smart caching system
* Performance monitoring
* Database cleanup tools
* Weekly optimization scheduling

**Pro Version:**
Upgrade to [Woo Performance Optimizer Pro](https://yourwebsite.com/woo-performance-optimizer-pro/) for advanced features like:
* Daily optimization scheduling
* Image optimization
* CDN integration
* Advanced caching
* Priority support

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woo-performance-optimizer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the settings under WooCommerce > Performance.
4. The plugin will start optimizing your store automatically.

== Frequently Asked Questions ==

= What does this plugin optimize? =

The plugin optimizes database queries, implements caching strategies, cleans up unnecessary data, and monitors performance.

= Do I need WooCommerce for this plugin? =

Yes, this plugin is specifically designed for WooCommerce stores.

= Will this plugin work with my theme? =

The plugin is designed to work with any WordPress theme, but results may vary depending on how your theme is coded.

= How often should I run optimizations? =

The Lite version allows weekly optimizations. For daily optimizations, consider upgrading to the Pro version.
